# forecast_LSTM

This program is meant to perform a time series prediction on temperature. Test data in the sqlite3 file `database.db`

### Prerequisites

- Python3
- pip
- PyTorch
- Pandas
- Tqdm

## Usage

First, `cd` to directory and then run either `python3 train.py` or `python3 train2.py`.
